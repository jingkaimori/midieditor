ProtocolEntry * SizeChangeTool::copy ( )
virtual

copies the ProtocolEntry.

copy() should save the ProtocolEntrys state so that it will be able to reload his old state in reloadState(). All important data has to be saved to the returned object; there is no need to save Layoutinformation because the program will relayout after every call of the protocol.
