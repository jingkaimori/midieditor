Detailed Description

ProtocolStep represents a Step in the Protocol, containing one ore more ProtocolItems.

Every ProtocolStep has a Stack with ProtocolItems. This items represent a single Action in the Protocol. A new Item can be added with addItem(). Calling releaseStep() will release all single Items and copy them to another ProtocolStep. This ProtocolStep is returned and represents the undo Action of this ProtocolStep. This is used to put a released undo-action onto the redo-stack of a protocol.
