Detailed Description

Every MidiChannel represents an Instrument.

In this program there are 18 MidiChannel: the 16 MidiChannels a MidiFile can describe, the Channel saving general Events (like PitchBend), one channel for TempoChangeEvents and one for TimeSignatureEvents.

Every channel can be mutes, visible in the MatrixWidget or playing alone (solomode).

Every Channel saves his own Events in a QMultiMap of the format miditick, Event.
