Introduction

MidiEditor is a free software providing an interface to edit, record, and play Midi data. The editor is able to open existing Midi files and modify their content. New files can be created and the user can enter his own composition by either recording Midi data from a connected Midi device (e.g., a digital piano or a keyboard) or by manually creating new notes and other Midi events. The recorded data can be easily quantified and edited afterwards using MidiEditor.

This manual will provide an overview over the many different features of MidiEditor. However, as it would go beyond the scope of this manual to discuss the foundations of Midi, only a quick overview will be provided here.
